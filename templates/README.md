# Templates

Store reusable LaTeX templates here.

Rules:

- One template per subdirectory.
- Template names should be lowercase and hyphen-separated.
- Templates are storage only and are not compiled automatically by GitHub Actions.
- To use a template, copy it into `incoming/<project-name>/` and rename the entry file to `main.tex` if needed.
